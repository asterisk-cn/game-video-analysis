# ゲームプレイデータ可視化Webアプリ

Unityゲームのプレイデータと動画を同期して可視化するReact + TypeScriptアプリケーション

## 機能

- **動画再生**: ゲームプレイ動画の再生
- **データ同期**: 動画の再生時刻に応じたパラメータ表示
- **データ駆動型可視化**: JSONから自動的に変数を抽出してグラフ表示
- **変数選択**: 表示する変数を個別に選択可能
- **イベント表示**: タイムライン上にイベントをドットで表示（イベントごとに選択可能）
- **インタラクティブ操作**: グラフをクリックして該当時刻にジャンプ
- **現在位置表示**: 動画再生位置をグラフ上に表示（表示/非表示切り替え可能）
- **GZIP対応**: 圧縮データファイルの自動解凍

## セットアップ

### 1. 依存パッケージのインストール

```bash
pnpm install
```

### 2. 開発サーバーの起動

```bash
pnpm dev
```

ブラウザで `http://localhost:5173` にアクセスします。

### 3. ビルド

```bash
pnpm build
```

ビルド成果物は `dist/` ディレクトリに出力されます。

### 4. プレビュー

```bash
pnpm preview
```

## 使用方法

1. **動画ファイルを選択**
   - 対応形式: MP4, WebM, OGG など（ブラウザがサポートする動画形式）

2. **データファイルを選択**
   - 対応形式: `.json` または `.json.gz`
   - Unityアセットから出力されたセッションデータ

3. **再生と分析**
   - ファイルをアップロードすると、ファイル選択UIは自動的に非表示になります
   - 動画を再生すると、現在時刻のパラメータが自動更新されます
   - グラフをクリックすると、その時刻に動画がジャンプします
   - 赤い破線が現在の再生位置を示します（表示/非表示切り替え可能）

4. **変数の選択**
   - 左側のパネルから表示する変数を選択できます
   - 最低1つの変数は常に選択状態になります

5. **イベントの選択**
   - 左側のパネルからイベントごとに表示/非表示を切り替えられます
   - イベントは選択されている変数の最大値の上にピンク色のドットで表示されます
   - ドットにホバーすると、イベント名と値が表示されます

6. **セッション情報**
   - 画面下部のセッション情報パネルは折りたたみ可能です
   - 矢印アイコンをクリックして表示/非表示を切り替えられます

## データ形式

アプリケーションは以下の形式のJSONデータを期待します：

```json
{
  "sessionInfo": {
    "generalInfo": {
      "sessionId": "uuid",
      "gameTitle": "ゲーム名",
      "startTime": "ISO8601形式",
      "endTime": "ISO8601形式"
    },
    "playerAttributes": {
      "age": 25,
      "skillLevel": "intermediate"
    },
    "gameSettings": {
      "difficulty": "normal",
      "soundEnabled": true
    }
  },
  "dataPoints": [
    {
      "timestamp": 0.0,
      "value": {
        "score": 0,
        "health": 100
      },
      "event": {
        "start": "game"
      }
    },
    {
      "timestamp": 1.0,
      "value": {
        "score": 10,
        "health": 100
      },
      "event": null
    },
    {
      "timestamp": 5.0,
      "value": {
        "score": 100,
        "health": 80
      },
      "event": {
        "itemCollected": "item1"
      }
    }
  ]
}
```

### データ構造の詳細

- **sessionInfo**: セッションメタデータ（任意の階層構造をサポート）
  - トップレベルのキーごとにセクション分けして表示

- **dataPoints**: 各データポイント
  - **timestamp**: 秒単位の経過時間（数値）
  - **value**: 任意の数値パラメータを含むオブジェクト
    - グラフに表示される変数はすべてこの`value`オブジェクト内に配置
  - **event**: イベント情報（オブジェクト形式、またはnull）
    - キーと値のペアでイベントを記録
    - 例: `{"itemCollected": "item1"}`, `{"enemyDefeated": "boss1"}`
    - グラフ上部にドット表示され、ホバーで詳細を確認可能

## データの追加

アプリケーションは完全にデータ駆動型のため、JSONの`value`オブジェクトに新しいパラメータを追加するだけで、自動的にグラフに表示されます。コードの変更は不要です。

```json
{
  "timestamp": 1.0,
  "value": {
    "score": 10,
    "health": 100,
    "customParam1": 50,
    "customParam2": 75
  },
  "event": null
}
```
